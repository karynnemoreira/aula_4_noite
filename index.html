Operadores lógicos

São usados comparando valores boleanos.

  • && (and) 'Ê' Seu resultado é true se ambos os valores forem verdadeiros
  • || (or) 'OU' Se um valou OU o outro valor é verdadeiro.
  • ! (not) Inverte o valor que é dado à ele

Loops: fazer a mesma coisa repetidas vezes

Operadores de atribuição

  • = atribuição de valor
  • + atribui concatenação
  • += atribuição de adição
  • - atribui uma subtração
  • -= atribuição de subtração
  • * multiplicação *= atribuição de multiplicação
  • / divisão /= atribuição de divisão
  • ++ atribuição de incremento. Operador unário. Adiciona 1 ao seu valor.
  • -- atribuição de decremento. Operador unário. Remove 1 do seu valor
